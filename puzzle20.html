Puzzle 20: Bytecode Interpreter

A custom bytecode interpreter executes the following instruction set. Decode the bytecode stream to find the output.

Bytecode Instruction Set:
  01 XX    = LOAD XX      (Load value XX into register A)
  02 XX    = XOR XX       (XOR register A with XX)
  03       = PRINT        (Print register A as ASCII)
  04 XX    = ADD XX       (Add XX to register A)

Initial State: A = 0

Bytecode Stream:
01 42 04 20 03 01 55 02 20 03 01 47 04 20 03

What three-character string does the program output?

Execute: LOAD, ADD, PRINT, LOAD, XOR, PRINT, LOAD, ADD, PRINT. Track register A through each operation.
