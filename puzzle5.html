Puzzle 5: Binary Stream

A raw binary bitstream was intercepted from a data transmission. The bits are not byte-aligned - you need to split them into 8-bit chunks manually.

Each 8 bits represents one ASCII character. Read from left to right.

Binary Stream:
011000110110100001101001011011010110101101101001011011010010000100100001

Split into groups of 8 bits, then convert each byte to ASCII. First byte: 01100011 = 0x63
