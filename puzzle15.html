Puzzle 15: Stripped Symbols

A binary was stripped of all symbol names. Only the operations remain. Trace through the logic to find the output.

var_1 = 0x42;
var_2 = 0x55;
var_3 = 0x47;

var_1 = var_1 + 0x20;    // var_1 = 0x62
var_2 = var_2 + 0x10;    // var_2 = 0x65
var_3 = var_3 + 0x20;    // var_3 = 0x67

var_4 = var_1;           // var_4 = 0x62
var_5 = var_2;           // var_5 = 0x65
var_6 = var_3;           // var_6 = 0x67

output(var_4);           // Output: ?
output(var_5);           // Output: ?
output(var_6);           // Output: ?

What three-character string does the program output?
