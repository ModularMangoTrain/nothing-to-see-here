Puzzle 13: Control Flow Flattening

A program's control flow was flattened - the blocks are shown out of order. Reconstruct the correct execution order.

Each block has a number. Follow the jumps to determine the execution sequence, then read the characters in order.

Block 3:
  char = 'p'
  goto Block 1

Block 1:
  char = 'i'
  goto Block 4

Block 5:
  char = 'a'
  goto END

Block 2:
  char = 'z'
  goto Block 5

Block 4:
  char = 'z'
  goto Block 2

START: goto Block 3
END: program terminates

What word is spelled by reading the characters in execution order?

Start at Block 3 and follow the goto statements in order.
