Puzzle 10: Custom VM

A simple virtual machine has only 4 instructions and one register (A). Execute the program mentally.

Instruction Set:
  INC   - Increment A by 1
  DEC   - Decrement A by 1
  XOR N - XOR A with value N
  OUT   - Output current value of A as ASCII

Initial State: A = 0x40 (64)

Program:
  INC           ; A = 0x41
  INC           ; A = 0x42
  INC           ; A = 0x43
  OUT           ; Output: 'C'
  DEC           ; A = 0x42
  XOR 0x20      ; A = A XOR 0x20
  OUT           ; Output: ?
  INC           ; A = A + 1
  INC           ; A = A + 1
  INC           ; A = A + 1
  OUT           ; Output: ?

What three-character string does the program output?

Track A through each instruction. Remember that OUT outputs the current value of A as ASCII.
