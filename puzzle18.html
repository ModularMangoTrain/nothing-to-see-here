Puzzle 18: Checksum Verification

A simple checksum algorithm is used to verify data integrity. Find the 4-letter word that produces the target checksum.

Checksum Algorithm:
  1. Start with sum = 0
  2. For each byte in input:
     - Add byte value to sum
     - XOR sum with 0xFF
  3. Return sum & 0xFF (last byte only)

Target Checksum: 0x42

Test these words:
  - "SHELDONCOOPER"
  - "LEONARDHOFSTADTER" 
  - "HOWARDWOLOWITZ"
  - "RAJESHKOOTHRAPPALI"

Which word produces checksum 0x42?

Calculate for each word using the algorithm. The answer is one of the test words provided.
