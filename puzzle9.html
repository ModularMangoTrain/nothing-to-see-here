Puzzle 9: Control Flow

A program uses conditional jumps to control execution. Follow the control flow to determine which value ends up in register A.

Instructions: CMP compares values. JZ jumps if zero (equal). JMP jumps unconditionally.

      MOV A, 0x41       ; A = 0x41
      CMP A, 0x41       ; Compare A with 0x41
      JZ label1         ; Jump to label1 if equal
      MOV A, 0x42       ; A = 0x42
      JMP end           ; Jump to end
label1:
      MOV A, 0x59       ; 
      CMP A, 0x50       ; 
      JZ label2         ; 
      ADD A, 0x06       ; 
      JMP end           ; 
label2:
      MOV A, 0x4E       ; 
end:
      ; Program ends here

What ASCII character is in register A at the end?

Start at the top. Follow each comparison and conditional jump carefully.
