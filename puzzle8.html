Puzzle 8: Stack Machine

A stack-based machine executes the following operations. The stack starts empty and grows upward.

PUSH adds a value to the top of the stack. POP removes the top value. Track the stack and find what's on top at the end.

Initial Stack: []

PUSH 0x48     ; Stack: [0x48]
PUSH 0x45     ; Stack: [0x48, 0x45]
PUSH 0x4C     ; Stack: [0x48, 0x45, 0x4C]
PUSH 0x4C     ; 
PUSH 0x4F     ;
POP           ; 
POP           ; 
PUSH 0x50     ; 
POP           ; 
POP           ; 
POP           ; 

Final Stack: [?]

What ASCII character is on top of the stack?

Follow each PUSH and POP operation carefully. Track what remains on the stack.
