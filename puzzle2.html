Puzzle 2: Byte Order

The system uses little-endian byte ordering. A 32-bit value was stored at address 0x1000. What word does it represent?

Address   Hex Dump
--------  ------------------------------------------------
00001000  45 56 4F 4C 00 00 00 00 FF FF FF FF 12 34 56 78

Little-endian stores bytes in reverse order. Read the first 4 bytes backwards.
